# Aplicação utiliza Laravel

Leia a documentação [Laravel documentação](https://laravel.com/docs/11.x).

## Rodar aplicação

- Docker

```bash
# docker-compose
docker-compose up -d
```
Acessar por: `http://localhost:8001`:

- Local

```bash
# composer
composer install

# php
php artisan serve --port=8001
```
Acessar por: `http://localhost:8001`:
